**A lightweight, local Gemini chat history viewer.**  
Visualize your Gemini chat history in a clear, tree-structured format without complex setups.

---

## 💎 What is this?
**RecollectViewer Lite** is a simplified, open-source tool designed for developers and power users who want to explore their **Gemini** chat history offline. It parses your local data and displays it in an intuitive tree view, allowing you to quickly navigate your past conversations.

This tool is built as a lightweight companion to the full **[RecollectViewer](https://www.yasui-kamo.com/labo/recollectviewer/)**.

## 🚀 Key Features
*   **🌳 Tree View Visualization:** Navigate your chat history structure effortlessly.
*   **🔒 100% Local & Private:** Your data stays on your machine. No cloud uploads.
*   **🌍 Global-Ready:** Designed to work smoothly across different system languages and date formats (ISO 8601).
*   **⚡ Lightweight:** Fast parsing with no unnecessary dependencies. Perfect for quick lookups.

## 🛠️ Technical Details
*   **Language:** C# (.NET Framework)
*   **Approach:** Built for high-performance parsing of local JSON files.
*   **Global Compatibility:** Handles path resolutions and date formatting (ISO 8601) regardless of your system's region settings.

## 📥 How to Use
1. Download the latest release from the [Releases page]([https://github.com/your-username/your-repo/releases](https://github.com/y-tomioka/RecollectViewerLite/releases)).
2. Run the `RecollectViewerLite.exe`.
3. Select your Google Takeout data folder.
4. Enjoy your chat history in a tree view!
